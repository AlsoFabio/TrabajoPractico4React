1)a)¿Qué es JSX?
    Es una extensión de sintaxis de javascript que nos permite mezclar codigo js y html, esta extensión nos permite escribir un código más limpio y sin tantas repeticiones.

b)¿Qué es un estado en React?
    El estado en React es el segundo tipo de dato que maneja esta librería de JavaScript. Mientras que las props son los datos que podemos pasarle a un componente o elemento React desde afuera, un estado se conforma por los datos internos que un componente puede manejar.

c)¿Qué es un componente?
    Los componentes permiten separar la interfaz de usuario en piezas independientes, reutilizables y pensar en cada pieza de forma aislada. Esta página proporciona una introducción a la idea de los componentes

d)¿Para qué sirve el hook useState?
    Sirve para manejar el estados de los elementos de un componente de manera que permite actualizar el estado del componente y solo renderiza la etiqueta html en donde es llamado.

e)¿Para qué sirve el hook useEffect?
    el useEffect nos sirve para realizar acciones que le pasemos como primer argumento (funcion) cada vez que se actualizan los datos que le pasemos como segundo argumento

f)¿Qué es el contexto en React?
    el contexto guarda datos para poder usarlos entre componentes sin tener que estar pasandolos uno por uno entre los distintos niveles de jerarquia de los componentes, sino llamarlos en donde los quieres utilizar
